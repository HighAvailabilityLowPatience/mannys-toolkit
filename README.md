mannys-toolkit

A curated repository of reusable code, infrastructure patterns, and automation utilities.

This repo acts as a personal engineering toolkit — a centralized place to store logic, modules, scripts, and infrastructure patterns that are useful across multiple projects.

Instead of rewriting common functionality, this repository allows me to quickly reuse and refactor proven code.

Purpose

 Engineers often rebuild the same functionality repeatedly across projects.

This repository exists to solve that problem by maintaining a library of reusable engineering components, including:

Infrastructure as Code modules

Automation playbooks

Logging and monitoring utilities

Common application logic

Small utilities and helper scripts

Patterns that can easily be refactored into larger systems

Over time this repository will grow into a personal DevOps and automation toolkit.
